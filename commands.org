#+title: Nix-config commands (Org Babel)
#+subtitle: Run maintenance commands here (TRAMP-ready)

This document defines the repository's maintenance commands as runnable
Org Babel blocks.

** Usage
Execute blocks with =C-c C-c= (org-babel-execute-src-block) in Emacs.
TRAMP-ready blocks support remote execution over SSH with sudo elevation.

* switch
Rebuild and activate NixOS configurations locally or remotely.
** local
#+name: rebuild-local
#+begin_src sh :dir (concat "/sudo:arthur@localhost:" (expand-file-name ".")) :async :results raw output drawer
nixos-rebuild switch --flake .#$(hostname) 2>&1
#+end_src

** local with substituters
#+begin_src sh :dir (concat "/sudo:arthur@localhost:" (expand-file-name ".")) :async :results raw output drawer
nixos-rebuild switch --flake .#$(hostname) --option extra-substituters "http://gmktec-k11.tail19d694.ts.net:5000?priority=30" 2>&1
#+end_src

* maintain server
Common gmktec-g3 home server maintenance commands.

** Build server configuration locally
Checks that the server system closure evaluates and builds before deploying.
#+name: build-gmktec-g3
#+begin_src sh :async :results raw output drawer
nix build .#nixosConfigurations.gmktec-g3.config.system.build.toplevel 2>&1
#+end_src

** Deploy current checkout to server
Build locally and activate on the server over SSH. This uses the committed or
working-tree flake from this checkout.
#+name: deploy-gmktec-g3
#+begin_src sh :async :results raw output drawer
NIX_SSHOPTS="-C" nixos-rebuild switch --flake .#gmktec-g3 --target-host gmktec-g3 --ask-sudo-password 2>&1
#+end_src

** Update flake inputs and deploy server
Update inputs in this checkout, build the server, then activate it remotely.
Review and commit the resulting flake.lock with jj afterwards.
#+name: update-deploy-gmktec-g3
#+begin_src sh :async :results raw output drawer
set -euo pipefail
nix flake update
nix build .#nixosConfigurations.gmktec-g3.config.system.build.toplevel
NIX_SSHOPTS="-C" nixos-rebuild switch --flake .#gmktec-g3 --target-host gmktec-g3 --ask-sudo-password 2>&1
#+end_src

* Garbage collect
Remove Nix store items older than 1 day, locally or on selected host.
#+name: gc-remote
#+begin_src elisp :var targethost=select-host
(let* ((current-host (system-name))
       (tramp-path (if (string= targethost current-host)
                       "/sudo::~/"
                     (format "/sshx:%s|sudo:%s:~/" targethost targethost))))
  (let ((default-directory tramp-path))
    (async-shell-command "nix-collect-garbage --delete-older-than 1d")))
#+end_src
